# Rule Sections

## Ordering and Impact Levels

| Section | Description | Typical Impact |
|---------|-------------|----------------|
| security | Authentication, authorisation, data protection | CRITICAL-HIGH |
| validation | Argument and return validators | HIGH |
| performance | Query optimisation, indexes, bounded results | HIGH-MEDIUM |
| architecture | Code organisation, function types, transactions | MEDIUM |
| code-quality | Promise handling, error patterns, table names | MEDIUM-LOW |
